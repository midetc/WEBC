# ФінТрекер - Додаток для обліку особистих витрат

Сучасний, мінімалістичний та практичний веб-додаток для обліку особистих витрат.

## Технічний стек

### Фронтенд
- React
- Next.js
- TypeScript
- Tailwind CSS

### Бекенд
- Python
- FastAPI
- Pydantic
- SQLAlchemy

### База даних та кешування
- PostgreSQL
- Redis

### ML-сервіс
- Python (scikit-learn)
- gRPC/REST API

### Аутентифікація та авторизація
- OAuth2 / JWT

## Структура проєкту

```
/
├── frontend/              # React + Next.js веб-клієнт
│   ├── src/
│   │   ├── components/    # React компоненти
│   │   ├── pages/         # Сторінки Next.js
│   │   └── styles/        # CSS стилі
│   └── ...
│
├── backend/               # FastAPI сервер
│   ├── app/
│   │   ├── api/           # Роути API
│   │   ├── models/        # Моделі даних
│   │   ├── services/      # Бізнес-логіка
│   │   └── ...
│   └── ...
│
└── ml/                    # Мікросервіс машинного навчання
    ├── src/
    │   ├── models/        # ML моделі
    │   ├── data/          # Обробка даних
    │   └── api/           # API для прогнозування
    └── ...
```

## Основні функції

1. **Аутентифікація**: реєстрація, вхід, вихід, опціонально 2FA
2. **CRUD витрат**: список, фільтрація, додавання, редагування, видалення
3. **Категорії**: створення, перейменування, вкладені підкатегорії, видалення
4. **Дашборд**: картки з підсумками, недавня активність
5. **Аналітика**: графіки трендів, порівняння з бюджетом
6. **Прогноз**: ML-прогноз на запит з кешуванням
7. **Профіль**: налаштування користувача, інтеграція OAuth
8. **Допомога та зворотній зв'язок**: FAQ, форма зворотного зв'язку

## Запуск проєкту для розробки

Для локальної розробки використовується Docker Compose:

```bash
# Клонувати репозиторій
git clone https://github.com/username/fin-tracker.git
cd fin-tracker

# Запустити Docker Compose
docker-compose up -d

# Фронтенд буде доступний на http://localhost:3000
# API бекенду доступно на http://localhost:8000
# Swagger документація на http://localhost:8000/docs
```
