# Spendio - Додаток для обліку особистих витрат

Мінімалістичний та зручний веб-додаток для обліку особистих витрат.

## Технічний стек

### Фронтенд
- React
- Next.js
- TypeScript
- Tailwind CSS

### Бекенд
- Python
- FastAPI
- SQLAlchemy
- Pydantic

## Функціональність

1. **Розміщення та стилізація елементів інтерфейсу**
   - Мінімалістичний, адаптивний дизайн з Tailwind CSS
   - Зручний інтерфейс для перегляду та додавання витрат
   - Категоризація витрат з легким управлінням

2. **Проектування та реалізація моделей даних для збереження інформації**
   - Простота та ефективність моделей даних
   - Основні сутності: Expense (витрата) та Category (категорія)

## Структура проєкту

```

└── backend/               # FastAPI сервер

    │   ├── db/            # Робота з базою даних
    │   └── ...
    └── ...
```

## Основні функції

1. **Огляд витрат**: перегляд усіх витрат, фільтрація за датою та категорією
2. **Додавання витрат**: створення нових записів про витрати з описом, сумою та категорією
3. **Категорії**: керування категоріями витрат (створення, видалення)
4. **Аналітика**: загальна сума витрат

## Запуск проєкту для розробки

```bash
# Клонувати репозиторій
git clone https://github.com/midetc/WEBC.git
cd WEBC

# Запуск бекенду
cd backend
pip install -r ../requirements.txt
python ../run_backend.py

# Запуск фронтенду (в іншій консолі)
cd frontend
npm install
npm run dev

# Фронтенд буде доступний на http://localhost:3000
# API бекенду доступно на http://localhost:8000
```
